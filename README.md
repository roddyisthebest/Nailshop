## 📱 앱 소개 

전국 모든 네일샵을 소개합니다!

유저들의 찜한 순서, 리뷰 평점으로 좋은 서비스를 제공하는 네일샵을 예약할 수 있어요! (전화,문자,카카오)

네일샵의 위도, 경도에 따라 가까운 순으로 네일샵 정보를 열람할 수 있어요!
